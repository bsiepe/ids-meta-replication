Data and code for A systematic Review and Bayesian Meta-Analysis of the Acoustic Features of Infant-Directed Speech. Pre-print here: https://psyarxiv.com/vn62t/

This OSF repository consists of the following:
	- Datasets used in the study,
	- Code in .Rmd format,
	- A knitted version of the code in .pdf format.

Note that the code was compiled on an Ubuntu virtual machine and requires 12 GB of free space to run. It will take approximately 36 hours to run with 64 cores and 188 GB RAM.
